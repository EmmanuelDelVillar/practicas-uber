Arduino Electrónica Introducción

- Resumen -

En el proceso de aprendizaje y desarrollo de habilidades con el Atiny 85, aprovecharemos la versatilidad de un Arduino uno como programador. Esta estrategia nos permitirá abordar una serie de programas básicos como punto de partida para familiarizarnos con este microcontrolador. Con el tiempo y la práctica, tendremos la capacidad de abordar desafíos más complejos, incorporando código de mayor sofisticación y funcionalidad en nuestros proyectos.

- Introducción -

El ATtiny85 es un microcontrolador de bajo costo y bajo consumo de energía fabricado por Atmel (ahora parte de Microchip Technology). Es parte de la familia ATtiny de microcontroladores y se utiliza comúnmente en proyectos electrónicos para realizar tareas de control y procesamiento de datos debido a su pequeño tamaño y eficiencia energética.

Un programador es un dispositivo o software que se utiliza para cargar o grabar código o firmware en microcontroladores, microprocesadores, memorias EEPROM y otros tipos de chips programables. Su función principal es transferir el programa o datos binarios desde un ordenador o una fuente de datos hacia el dispositivo objetivo.

Ejercicio 13: Attiny Contador + 3 LEDs

// Codigo Attiny85
//
int contador=0;
void setup()
{
pinMode(0, OUTPUT);
pinMode(1, OUTPUT);
pinMode(2, OUTPUT);
pinMode(3, INPUT);
}
void loop()
{
if(digitalRead(3)==HIGH){
contador++;
}
if(contador>3){
contador=0;
} if(contador==0){ digitalWrite(0, LOW);
digitalWrite(1, LOW);
digitalWrite(2, LOW);
}
else if(contador==1){
digitalWrite(0, HIGH);

digitalWrite(2, LOW);
}
else if(contador==2){
digitalWrite(0, HIGH);
digitalWrite(1, HIGH);
digitalWrite(2, LOW);
}
else if(contador==3){
digitalWrite(0, HIGH);
digitalWrite(1, HIGH);

}

}

Ejercicio 14: Attiny Contador + - 3 LEDs

// Codigo Attiny85
//
int contador=0;
void setup()
{
pinMode(0, OUTPUT);
pinMode(1, OUTPUT);
pinMode(2, OUTPUT);
pinMode(3, INPUT);
pinMode(4, INPUT);
}
void loop()
{
if(digitalRead(3)==HIGH){
contador++;
}
if(digitalRead(4)==HIGH){
contador--;
}
if(contador>3){
contador=3;
}
if(contador/0){

}

digitalWrite(2, LOW);
}
else if(contador==1){

digitalWrite(2, LOW);
}
else if(contador==2){
digitalWrite(0, HIGH);
digitalWrite(1, HIGH);
digitalWrite(2, LOW);
}
else if(contador==3){
digitalWrite(0, HIGH);

}
delay(500);
}
