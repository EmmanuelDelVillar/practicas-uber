Arduino Electrónica Introducción

- Resumen -

Esta Practica es la continuacion de la practica anterior, se realizaran varios ejercicios básicos con un Arduino uno, la finalidad de estos ejercicios es el recordar o aprender el funcionamiento básico de la programación de un Arduino para poder implementarlo para proyectos futuros.

- Introducción -

Arduino es una plataforma de software y hardware libres basada en una placa que incluye circuitos electrónicos con entradas y salidas, analógicas y digitales, que conectan el mundo físico con el mundo virtual, a partir de la cual se pueden crear objetos electrónicos interactivos, como robots o sistemas domóticos.

El Arduino es una placa basada en un microcontrolador ATMEL. Los microcontroladores son circuitos integrados en los que se pueden grabar instrucciones, las cuales las escribes con el lenguaje de programación que puedes utilizar en el entorno Arduino IDE. Estas instrucciones permiten crear programas que interactúan con los circuitos de la placa.

Ejercicio 1: Servo 1

// Incluímos la librería para poder controlar el servo
#include /Servo.h/
// Declaramos la variable para controlar el servo
Servo servoMotor;
void setup()
{
// Iniciamos el monitor serie para mostrar el resultado
Serial.begin(9600);
// Iniciamos el servo para que empiece a trabajar con el pin 9
servoMotor.attach(9);
}
void loop()
{
// Desplazamos a la posición 0º
servoMotor.write(0);
// Esperamos 1 segundo
delay(1000);
// Desplazamos a la posición 90º
servoMotor.write(90);
// Esperamos 1 segundo
delay(1000);
// Desplazamos a la posición 180º
servoMotor.write(180);
// Esperamos 1 segundo
delay(1000);
}

Ejercicio 2: Servo 2

// Incluímos la librería para poder controlar el servo
#include /Servo.h/
// Declaramos la variable para controlar el servo
Servo servoMotor;
void setup()
{
// Iniciamos el monitor serie para mostrar el resultado
Serial.begin(9600);
// Iniciamos el servo para que empiece a trabajar con el pin 9
servoMotor.attach(9);
}
void loop()
{
// Vamos a tener dos bucles uno para mover en sentido positivo y otro en sentido negativo
// Para el sentido positivo
for (int i = 0; i = 180; i++)
{ // Desplazamos al ángulo correspondiente
servoMotor.write(i);
// Hacemos una pausa de 25ms
delay(25);
}
// Para el sentido negativo
for (int i = 179; i > 0; i--)
{
// Desplazamos al ángulo correspondiente
servoMotor.write(i);
// Hacemos una pausa de 25ms
delay(25);
}
}

Ejercicio 7: Servos Independientes y leds

// C++ code
#include /Servo.h/
int valor1; //variable que almacena la
//lectura analógica1
int valor2; //variable que almacena la
//lectura analógica2
int pos1; //Variable que almacena la posicion del servo1
int pos2; //Variable que almacena la posicion del servo2
//Le decimos al codigo que va a existir un servo
//llamado my servo
Servo myservo1;
Servo myservo2;
void setup()
{
pinMode(12, OUTPUT); //LED1
pinMode(11, OUTPUT); //LED2
//Le decimos al codigo donde esta conectado el servo 1
myservo1.attach(9);
//Le decimos al codigo donde esta conectado el servo 2
myservo2.attach(2);
}
void loop()
{
// leemos el valor de potenciometro1
valor1 = analogRead(A0);
// leemos el valor de potenciometro2
valor2 = analogRead(A1);
//Convertimos el valor del potenciometro a una
//que entienda el servo
pos1 = map(valor1, 0, 1023, 0, 180);
pos2 = map(valor2, 0, 1023, 0, 180);
//Mandamos la posicion al servo 1
myservo1.write(pos1);
if (pos1 = 10){
digitalWrite(12, HIGH); //Encendemos led1
} else if (pos1 >= 170){
digitalWrite(11, HIGH); //Encendemos led1
} else
{
digitalWrite(12, LOW); //Encendemos led1
digitalWrite(11, LOW); //Encendemos led1
}
//Mandamos la posicion al servo 2
myservo2.write(pos2);
//esperamos un poco para que se mueva
delay(10);
}
