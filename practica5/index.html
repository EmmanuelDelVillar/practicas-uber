Arduino Electrónica Introducción Attiny

- Resumen -

En esta práctica conectare una pantalla oled al Attiny 85 utilizando el protocolo de comunicación I2C y mostrare el estado de un botón en tiempo real

- Introducción -

El protocolo I2C es un estándar de comunicación serial que utiliza dos líneas, SDA (Serial Data) y SCL (Serial Clock), para permitir la comunicación entre dispositivos electrónicos en un sistema. El dispositivo maestro inicia y controla la comunicación con dispositivos esclavos. La comunicación se basa en la transmisión de datos secuenciales y puede ser bidireccional, lo que significa que tanto el maestro como el esclavo pueden enviar y recibir datos. Esto permite que los dispositivos se comuniquen de manera efectiva para intercambiar información, como comandos, sensores y otros datos.

Una pantalla OLED es un tipo de pantalla que utiliza diodos orgánicos emisores de luz para mostrar imágenes. Es delgada, flexible y eficiente energéticamente, con la capacidad de mostrar colores vibrantes y negros profundos. Se usa en una amplia gama de dispositivos, desde teléfonos inteligentes hasta relojes inteligentes, y puede ser controlada mediante I2C para mostrar información y gráficos.
