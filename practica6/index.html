- Resumen -

En esta práctica se investigó los diferentes tipos de sensores que hay analógicos, digitales y inteligentes. Se creó una tabla con los diferentes modelos de sensores junto con su información correspondiente. Además se utilizaron dos sensores de cada tipo para usarlos de ejemplo en un circuito

- Introducción -

Los sensores son dispositivos que detectan cambios en el entorno físico o químico y convierten estos cambios en señales que pueden ser utilizadas en sistemas electrónicos, máquinas o por humanos. Tienen muchas aplicaciones, como medir variables como temperatura, presión y luz, controlar procesos industriales, habilitar funciones en dispositivos electrónicos de consumo, mejorar la seguridad en automóviles y en medicina, entre otros usos. Los sensores son esenciales para la recopilación de datos y la toma de decisiones en diversas áreas.

Algunas de las señales que ocupan los sensores son Analogica y digital. La señal analógica es una representación continua y variable de una magnitud física que cambia con el tiempo. Se utiliza para representar fenómenos como el sonido, la luz, la temperatura, etc., y puede tomar infinitos valores en un rango determinado. Se diferencia de las señales digitales, que son discretas y representan información en forma de 0 y 1.

VIDEOS DE PRACTICAS

SENSOR DE OBSTACULO

Conectaremos el sensor de obstáculo y activaremos un led si este nos detecta

SENSOR DE VIBRACION

Conectaremos el sensor de vibración y activaremos un led si este nos detecta

SENSOR DE TEMPERATURA

Conectaremos el sensor de temperatura y para poder ver los datos que arroja el sensor utilizaremos un Arduino para ver los datos en el monitor serial

SENSOR DE INTENCIDAD DE LUZ

Conectaremos el sensor de intensidad de luz y activaremos un led si este excede el parámetro establecido

SENSOR DE TEMPERATURA Y HUMEDAD

Conectaremos el sensor de temperatura y humedad y al igual que el sensor de temperatura, utilizaremos un Arduino para visualizar los datos en el monitor serial

SENSOR DE DISTANCIA

Conectaremos el sensor de distancia y activaremos una serie de leds si este excede los parámetros establecidos

- Conclusiones -

Gracias a esta practica identifique los diferentes tipos de sensores que hay y como estos se pueden usar, dependiendo de las necesidades del proyecto a realizar, además de que existen algunos sensores que utilizan protocolos de comunicación para facilitar su uso.
